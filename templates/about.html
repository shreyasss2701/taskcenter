The task Management System

TaskCenter is a Django-based task management system built using the MVT architecture to ensure a clean and scalable structure. It allows users to perform CRUD operations on tasks, including creation, updating, and assignment with proper user relationships. The system implements role-based access control using Django’s authentication framework, enabling assigner and assignee functionalities. Tasks are managed with defined priority levels and status tracking to monitor progress efficiently. The application uses Django ORM for optimized database operations, ensuring data integrity and maintainability.
